# Awesome Books

Awesome Books is a simple project that allows users to add and remove books from a list. It is built using HTML, CSS, and JavaScript.

## Features

- Add a new book with a title and author.
- Remove a book from the list.
- Data is stored in the browser's local storage, so the list persists even after refreshing the page.

## Getting Started

To get a local copy up and running, follow these steps:

### Prerequisites

- A modern web browser.
- Basic knowledge of HTML, CSS, and JavaScript.

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/awesome-books.git
    ```
2. Navigate to the project directory:
    ```bash
    cd awesome-books
    ```
3. Open the `index.html` file in your browser to view the app.

## Usage

1. Enter the title and author of a book in the input fields.
2. Click the "Add" button to add the book to the list.
3. Click the "Remove" button next to a book to delete it from the list.
